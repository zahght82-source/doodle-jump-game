# Doodle Jump Clone (C++ / SFML)

Final project for the Advanced Programming course, University of Tehran.

## Overview

A C++ recreation of the classic Doodle Jump platformer, built with the SFML graphics library, focused on solid object-oriented design rather than just getting the game running.

## Architecture

- **Full OOP design** — built around polymorphism and templates rather than one monolithic game loop.
- **Design patterns** applied throughout the codebase to keep game objects (player, platforms, obstacles, etc.) decoupled and extensible.
- **Multi-folder source structure** separating game logic, rendering, and entities instead of a flat file layout.
- Developed with a **Git/GitHub workflow** on a Windows/MSYS2 UCRT64 environment.

## Tech Stack

C++, SFML
